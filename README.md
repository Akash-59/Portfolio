# Akash Kumar - Portfolio Website

![Portfolio Screenshot](https://via.placeholder.com/800x400.png?text=Portfolio+Screenshot)

🌟 **Professional portfolio showcasing my skills, projects, and experience as a Full Stack Developer.** This responsive website features modern design elements and smooth animations to highlight my technical capabilities and creative approach to web development.

## 🚀 Features
- **Modern UI/UX Design** - Elegant dark theme with golden accents and gradient effects
- **Fully Responsive** - Works flawlessly on all devices from mobile to desktop
- **Interactive Elements**:
  - Typing animation for dynamic text effects
  - Particle background with floating elements
  - Smooth hover animations and transitions
  - Scroll-activated animations (using AOS library)
- **Project Showcase** - Highlighting my best work with live demos
- **Contact Section** - Easy access to my professional information

## 🛠️ Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript
- **Frameworks**: Bootstrap 5
- **Libraries**: 
  - Font Awesome (Icons)
  - Animate On Scroll (AOS)
  - Typed.js (Typing animations)
- **Design**: Custom CSS with modern gradients and animations
